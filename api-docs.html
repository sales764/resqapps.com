üöÄ Getting Started

The RESQ+ API allows you to integrate emergency response services into your application. Access real-time emergency alerts, hospital data, and coordinate emergency responses.

üìã Prerequisites

  • Register for an API key at resqapps.com/api/register
  • Understand REST API principles
  • Have a valid use case for emergency services integration

Base URL

https://api.resqapps.com/v1

Quick Example

curl -X GET "https://api.resqapps.com/v1/alerts" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"

üîê Authentication

All API requests require authentication using an API key. Include your API key in the Authorization header.

API Key Authentication

Authorization: Bearer YOUR_API_KEY

üíª Code Examples

JavaScript (Node.js)

const axios = require('axios');

const API_KEY = 'your_api_key_here';
const BASE_URL = 'https://api.resqapps.com/v1';

async function createEmergency() {
  try {
    const response = await axios.post(`${BASE_URL}/emergency`, {
      type: 'medical',
      severity: 'high',
      location: {
        latitude: 13.7563,
        longitude: 100.5018
      },
      description: 'Heart attack victim',
      contactPhone: '+66812345678'
    }, {
      headers: {
        'Authorization': `Bearer ${API_KEY}`,
        'Content-Type': 'application/json'
      }
    });
    
    console.log('Emergency created:', response.data);
  } catch (error) {
    console.error('Error:', error.response.data);
  }
}

createEmergency();

Python

import requests

API_KEY = 'your_api_key_here'
BASE_URL = 'https://api.resqapps.com/v1'

def create_emergency():
    headers = {
        'Authorization': f'Bearer {API_KEY}',
        'Content-Type': 'application/json'
    }
    
    data = {
        'type': 'medical',
        'severity': 'high',
        'location': {
            'latitude': 13.7563,
            'longitude': 100.5018
        },
        'description': 'Heart attack victim',
        'contactPhone': '+66812345678'
    }
    
    response = requests.post(f'{BASE_URL}/emergency', json=data, headers=headers)
    
    if response.status_code == 201:
        print('Emergency created:', response.json())
    else:
        print('Error:', response.json())

create_emergency()
