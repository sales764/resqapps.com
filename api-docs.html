🚀 Getting Started

This page describes a planned RESQ+ API surface for evaluation and future partner discussions. The current mobile app does not rely on this public REST documentation for production operation, and this API should be treated as preview-only.

📋 Prerequisites

  • Understand that this documentation is for preview and planning purposes
  • Review REST and JSON integration patterns
  • Contact RESQ+ directly before assuming any endpoint is available for production use

⚠️ Preview Status

  • No public self-service registration flow is currently published
  • Endpoint shapes shown below are illustrative and may change
  • Do not build production dependencies against this document without direct confirmation

Planned Base URL

https://api.resqapps.com/v1

Draft Example

# Preview only - illustrative request format
curl -X GET "https://api.resqapps.com/v1/alerts" \
  -H "Authorization: Bearer PREVIEW_TOKEN" \
  -H "Content-Type: application/json"

🔐 Authentication

Authentication details are not currently offered as a public self-service workflow. Any future access would be provisioned directly by RESQ+ for approved preview or partner use cases.

Planned Authorization Header

Authorization: Bearer PREVIEW_TOKEN

⚠️ Security

  • Never expose preview credentials in client-side code
  • Treat all credentials and schemas on this page as non-final
  • Use HTTPS for all requests

❌ Error Handling

Preview integrations should expect schema and error behavior to evolve. Status codes and payloads below are illustrative rather than contractual.

HTTP Status Codes

Code Description
200 Success
201 Created
400 Bad Request
401 Unauthorized
403 Forbidden
404 Not Found

💻 Code Examples

The examples below show draft request structure only. They are not a guarantee that these endpoints are currently active for public production use.

JavaScript (Node.js)

const axios = require('axios');

const API_KEY = 'preview_token_here';
const BASE_URL = 'https://api.resqapps.com/v1';

async function previewEmergencyDraft() {
  try {
    const response = await axios.post(`${BASE_URL}/emergency`, {
      type: 'medical',
      severity: 'high',
      location: {
        latitude: 13.7563,
        longitude: 100.5018
      },
      description: 'Heart attack victim',
      contactPhone: '+66812345678'
    }, {
      headers: {
        'Authorization': `Bearer ${API_KEY}`,
        'Content-Type': 'application/json'
      }
    });
    
    console.log('Preview response:', response.data);
  } catch (error) {
    console.error('Preview error:', error.response?.data || error.message);
  }
}

previewEmergencyDraft();

Python

import requests

API_KEY = 'preview_token_here'
BASE_URL = 'https://api.resqapps.com/v1'

def preview_emergency_draft():
    headers = {
        'Authorization': f'Bearer {API_KEY}',
        'Content-Type': 'application/json'
    }
    
    data = {
        'type': 'medical',
        'severity': 'high',
        'location': {
            'latitude': 13.7563,
            'longitude': 100.5018
        },
        'description': 'Heart attack victim',
        'contactPhone': '+66812345678'
    }
    
    response = requests.post(f'{BASE_URL}/emergency', json=data, headers=headers)
    
    if response.ok:
        print('Preview response:', response.json())
    else:
        print('Preview error:', response.text)

preview_emergency_draft()
