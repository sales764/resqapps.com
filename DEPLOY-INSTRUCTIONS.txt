================================================================================
TROUBLESHOOTING
================================================================================

Problem: "404 - File not found"
Solution: 
- Make sure index.html is in root folder
- Wait 5 minutes after enabling Pages
- Clear browser cache

Problem: "Custom domain not working"
Solution:
- DNS takes 24-48 hours to work
- Check dnschecker.org
- Verify DNS records are correct
- Make sure you typed resq.com (not www.resq.com) in GitHub

Problem: "Images not loading"
Solution:
- Make sure /images folder was uploaded
- Check file names match exactly (case-sensitive)

Problem: "Can't enable HTTPS"
Solution:
- Wait for DNS to fully propagate first
- Remove and re-add custom domain
- Wait 10 minutes and try again
